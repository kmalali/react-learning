# Exercises

## Table with weather information for cities
Create a table which displays weather information like minimum temperature, maximum temperature etc. for a few cities.

## Profile widget
Write a Profile component that displays a profile of a person. It accepts an image URL and name of person as props. The component can have arbitrary structured content that gets displayed as such on the profile.
